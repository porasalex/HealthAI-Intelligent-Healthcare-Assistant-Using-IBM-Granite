HealthAI: Intelligent Healthcare Assistant

Powered by IBM Granite & Watson Machine Learning

Category: Cloud Application Development

Skills Required: Python, IBM Cloud, Scikit-Learn

Project Features

  • Patient Chat: Ask health-related questions and receive empathetic, AI-generated responses.
  • Disease Prediction: Input symptoms to get potential diagnoses with likelihood scores.
  • Treatment Plans: Generate personalized medical recommendations based on user data.
  • Health Analytics: Visualize and monitor vital signs like heart rate and blood pressure.

Technology Stack

  • IBM Granite-13b-instruct-v2 for generative AI
  • IBM Watson Machine Learning for backend intelligence
  • Streamlit for user interface
  • Secure API key management and ethical data handling

Example Scenarios

  • User inputs symptoms like headache and fatigue → receives condition predictions and next steps.
  • User requests treatment for a known condition → receives a tailored plan with medications and lifestyle tips.
  • User tracks health metrics → receives insights and alerts via Health Analytics dashboard.
  • User asks a general health question → gets a clear, informative response with medical context.
